## How to Run
1. Clone the repo using:  
   install Visual Studio (recommend VS2022)  
   ```sh
    git clone https://github.com/ucsd-cse125-sp23/group-3.git
    ```
2. Run the server:  
   Double click server.sln  
   Run Local Windows Debugger
   
3. Run the client:  
   Double click BlinderEngine.sln  
   Run Local Windows Debugger
